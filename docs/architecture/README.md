## Template

New ADRs should follow this structure:

```markdown
# ADR-NNN: Title

**Status:** Proposed | Accepted | Deprecated | Superseded  
**Date:** YYYY-MM-DD  
**Deciders:** [list]

## Context
## Decision
## Consequences
## Alternatives Considered
```
