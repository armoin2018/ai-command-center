## Workflow

### Phase 1: Assess
1. Determine Bootstrap version and project setup (CDN vs npm)
2. Analyze layout requirements and responsive breakpoints
3. Identify needed components and JavaScript plugins

### Phase 2: Implement
1. Structure HTML with semantic elements and Bootstrap grid
2. Apply Bootstrap utility classes following utility-first approach
3. Customize theme via SCSS variables and maps
4. Configure JavaScript plugins with proper initialization
5. Ensure all interactive elements are keyboard accessible

### Phase 3: Validate
1. Test responsive behavior across all Bootstrap breakpoints (xs, sm, md, lg, xl, xxl)
2. Verify ARIA attributes and keyboard navigation
3. Check dark mode compatibility
4. Validate against Bootstrap best practices
