## Workflow

### Phase 1: Assess
1. Analyze the styling requirements and target browsers
2. Determine layout approach (Grid, Flexbox, or hybrid)
3. Identify responsive breakpoints and design tokens

### Phase 2: Implement
1. Apply modern CSS features with appropriate fallbacks
2. Use logical properties for internationalization support
3. Implement responsive patterns with mobile-first approach
4. Add accessibility features (focus-visible, prefers-reduced-motion, prefers-color-scheme)

### Phase 3: Validate
1. Verify cross-browser compatibility
2. Check responsive behavior across breakpoints
3. Validate accessibility of styled elements
4. Audit performance (specificity, unused styles, render-blocking)
