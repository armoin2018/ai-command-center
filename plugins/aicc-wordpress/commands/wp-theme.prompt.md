## Task

Manage WordPress themes:

### Actions

1. **List** — List installed themes with active status
2. **Search** — Search the WordPress theme directory by keyword
3. **Install** — Install a theme by slug
4. **Activate** — Activate an installed theme
5. **Get** — Get details of the currently active theme
6. **Customize** — Describe recommended customizations based on the active theme's capabilities
